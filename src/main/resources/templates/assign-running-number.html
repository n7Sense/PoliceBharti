Assign Running Number

Select a batch, lock it, then assign running number by Application No.

Candidate Details
Lock batch first, then press Enter.
Instructions
  • Select the unlocked batch (B1, B2, ...).
  • Click Lock before entering Application No.
  • Candidate must have attendance=true and status=true.
  • Each batch assigns only 20 candidates, then it becomes full.
Use this if current batch is locked by another user or full.
